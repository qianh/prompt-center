## 📱 界面功能

### 1. 提示管理 (Prompts)
- **提示列表** - 卡片式布局，支持搜索和标签过滤
- **提示编辑器** - 富文本编辑，标签管理
- **版本历史** - 时间线展示，版本比较

- **Diff显示** - 清晰的变更标记
- **回滚功能** - 快速恢复到历史版本

### 3. 比较功能 (Comparisons)
- **比较仪表板** - 统计概览和性能指标
- **新建比较** - 直观的配置向导
- **结果展示** - 详细的执行结果和质量分析
- **重试机制** - 失败执行的重试功能

### 4. 系统导航
- **顶部导航** - 清晰的页面切换

## 🎨 设计系统

### 颜色规范
```css
:root {
  --primary: 221.2 83.2% 53.3%;     /* 主色调 - 蓝色 */
  --secondary: 210 40% 96%;         /* 次要色 - 浅灰 */
  --accent: 210 40% 96%;            /* 强调色 */
  --destructive: 0 84.2% 60.2%;     /* 危险色 - 红色 */
  --muted: 210 40% 96%;             /* 静音色 */
  --background: 0 0% 100%;          /* 背景色 */
  --foreground: 222.2 84% 4.9%;     /* 前景色 */
}
```

### 间距规范
- **xs**: 0.25rem (4px)
- **sm**: 0.5rem (8px)
- **md**: 1rem (16px)
- **lg**: 1.5rem (24px)
- **xl**: 2rem (32px)

### 字体规范
- **标题**: font-bold text-xl/lg/2xl
- **正文**: text-sm text-gray-600
- **说明**: text-xs text-gray-500
